SWM Data Visualizaton Samples

A brief selection of data dasbhoards built from WISKI and Python.

Let your data do the talking.

Below, you will find two samples of automated data reports that I built using the KIWIS web API for our WISKI database. KIWIS allows users to request WISKI data remotely. Data can be analyzed and shown to audiences in a way that is not possible using the standard WISKI interface (available only to trained SWM staff) or the standard, web-based WISKI WaterDataViewer page. Python, JavaScript, or R programming languages allow me to build custom programs so that—with the click of a button or the loading of a web page—users can see the data they need in an intuitive and interactive format.

The samples blow are only a glimpse of what is possible. Literally any of the data stored in WISKI can be exported, repackaged and visualized to tell a story. This flexibility allows decision-makers access to the data they need without requiring any understanding of WISKI or the underlying data structures. Once designed, they can be updated at the click of a button, so Monitoring staff don't have waste time with manual updates. With a little extra effort, the data can even be streamed live and published to the County's website (real-time, interactive flood data, perhaps?). Pretty neat, and a huge opportunity to improve SWM's data narrative.

Shellfish Growing Area Report

In this example, I plotted fecal coliform results against rainfall data so that the Shellfish Protection District stakeholders can see how runoff affects bacteria. Notice anything suspicious about the summer of 2018? And, since not everyone knows where we get our data, I included a map of the weather station and sampling location. Laslty, I included the results table to aid in interpretation. The graphics are interactive, so you can zoom in and out on the chart or the map. If you click on a row in the table, the corresponding data point is highlighted on the graph. These are all fairly simple customizations that allow the user to dive into the data and formulate thier own insights.

Clover Creek Fecal Coliform Report

Here, I generated a 'Categorical Heat Map' for past fecal coliform results from all 23 of the monitoring locations on Clover Creek and the North Fork. There is a logrithmic key to the right that gives gives the fecal result value categories. The colors themselves were chosen to be somewhat intuitive (and colorblind friendly!), but you can always hover over a point to see the exact count for a given station and date. The hope is that we can begin to spot trends, seasonalities, and hot-spots in a dataset that is otherwise extremely variable.
