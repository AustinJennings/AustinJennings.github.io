WISKI Data Visualization Samples

Below, you will find two samples of automated data reports that I built using the KIWIS web API for our WISKI database. KIWIS allows users to request WISKI data remotely. Data can be analyzed and shown to audiences in a way that is not possible using just the WISKI program (available only to trained SWM staff) or the standard, web-based WISKI WaterDataViewer page. Python, JavaScript, or R programming languages allow me to build custom programs so that—with the click of a button or the loading of a web page—users can see the data they need in an intuitive and interactive format. The samples blow are only a glimpse of what is possible, though. Literally any of the data stored in WISKI can be exported, repackaged and visualized to tell a story. This flexibility allows decision-makers access to the data they need without requiring any understanding of the underlying WISKI database. And better yet, the reports are automated snapshots, so Monitoring staff don't have to spend time manually updating them. With a little extra effort, the data can even be streamed live and published to the County's website (emergency flood updates, perhaps?). Pretty neat, and a huge opportunity to improve SWM's data narrative.
