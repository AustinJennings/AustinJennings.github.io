Embedded Maps and Web Building

My first forray into coding and website construction.

Web GIS Lab 1.1

For my Web-Based GIS course (TGIS503), I created this embedded Google Maps frame as part of my first lab. The purpose of this lab was to code, using only a text editor, a simple website with some basic interactive features. To do this, we learned some HTML, CSS, and JavaScript, then used FTP to upload it to our UW student hosting service. You can view my student website, but this site is due to expire in 2018.
